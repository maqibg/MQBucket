手动检测版本

要手动检测所有应用的最新版本，请在终端运行以下命令：

pwsh -File scripts/generate-status.ps1 -Token "your_token"

检测完成后，点击“重新加载数据”按钮刷新页面。
检测耗时取决于当前 bucket 数量与上游接口响应速度。
建议使用 Token 避免 API 限流（60次/小时 → 5000次/小时）
